# Cahaya Putra Mandiri Website

A modern, responsive company profile website developed for Cahaya Putra Mandiri to showcase services, products, and company information with a clean and professional user experience.

## Features

- Responsive Design
- Modern Landing Page
- Product Showcase
- Company Information
- Contact Section
- Smooth Scrolling
- Mobile Navigation

## Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)
